How to Use the Dropdown Manager Component

This guide will help you manage various dropdown items stored in your Firebase Firestore database using the Dropdown Manager Component.

Step-by-Step Instructions:

  1. Select a Dropdown Type:

    In the "Select Dropdown to Manage" section, choose the type of dropdown you want to manage by clicking on one of the radio buttons (e.g., Categories, States, Status, Profile Types, Product Types).

  2. View Existing Items:

    After selecting a dropdown type, a list of existing items for that dropdown will be displayed. Each item will have an input field and a "Delete" button.

  3. Update an Item:

    To update an item's name, simply modify the text in the input field. The change will be saved automatically when you click outside the input field.

  4. Delete an Item:

    To delete an item, click the "Delete" button next to the item. This will remove the item from the list and the database.

  5. Add a New Item:

    At the bottom of the list, there is an "Add Item" section.

    • Enter the name of the new item in the input field provided.
    • Click the "Add" button to save the new item to the database. The item will then appear in the list of existing items.

Example Usage:

  • Select "Categories" from the radio buttons.
  • View the current list of categories.
  • Edit the name of a category by typing a new name and clicking outside the input field to save.
  • Delete a category by clicking the "Delete" button next to it.
  • Add a new category by typing its name in the "New Item" field and clicking "Add."

Notes:

  • Ensure you have a stable internet connection as the component interacts with Firebase Firestore to fetch, update, and delete data.
  • Any changes you make (add, update, delete) are immediately reflected in the database.

This component provides a user-friendly interface to manage dropdown items efficiently, enhancing the overall usability of your application.
