Troubleshooting

Encounter an issue? Our troubleshooting guide provides solutions to common problems and steps to resolve them efficiently. If you need further assistance, you can toggle data into JSON view to provide our support team with detailed information.

Common Issues and Solutions

  • Problem: Unable to Log In

    Description: You're unable to log in despite entering the correct credentials.

    Solution: Ensure your internet connection is stable. If the problem persists, try resetting your password or using the passwordless sign-in option. Make sure cookies and JavaScript are enabled in your browser settings.

  • Problem: Data Not Syncing

    Description: Changes made are not reflecting in real-time or data seems outdated.

    Solution: Refresh the page to see if the updates are reflected. Check if the auto-sync feature is enabled in your settings. If issues continue, restart the application.

  • Problem: Error Messages Displayed

    Description: Receiving error messages when attempting to submit data or complete actions within the application.

    Solution: Take note of the error code or message. Refer to our error code section for specific troubleshooting steps or contact support with the details.

  • Problem: Features Not Loading

    Description: Certain features or pages are not loading or are partially loading.

    Solution: Ensure your browser is up-to-date and supports the application. Clear your browser cache and cookies. If the issue persists, try accessing the application from a different browser or device.

  • Problem: Missing Data or Incorrect Data Displays

    Description: The data shown is incorrect or not up to date, possibly affecting reports or outputs.

    Solution: Verify the source of the data input for errors. If using automated data importing tools, check the import logs for errors. Use the 'Data Show in JSON View' feature to verify the integrity and correctness of data shown in the application.

Toggling Data to JSON View

If you encounter issues that require detailed diagnosis, you can toggle your current session data into JSON view, which can be copied and sent to our support team for further analysis.

  1. Navigate to the 'Settings' or 'Help' menu within the application.
  2. Select the 'Show Data in JSON View' option.
  3. Copy the JSON data displayed on the screen.
  4. Email the copied data to support with a brief description of the issue.

Glossary

Our application uses specific terms that may be unfamiliar. This glossary provides definitions of these terms to help you better understand the features and functionalities of our platform.

NLP (Natural Language Processing)
Technology that enables computers to understand and interpret human language the way it is spoken or written. Used in our platform to enhance searching, querying, and data analysis capabilities.
API (Application Programming Interface)
A set of rules and protocols for building and interacting with software applications. API allows different software systems to communicate with each other.
CSV (Comma-Separated Values)
A file format used to store data in a tabular form, such as spreadsheets or databases. Each line of the file is a data record, and each record consists of one or more fields separated by commas.
Blockchain
A system of recording information in a way that makes it difficult or impossible to change, hack, or cheat the system. A blockchain is essentially a digital ledger of transactions that is duplicated and distributed across the entire network of computer systems on the blockchain.
Dashboard
A user interface that organizes and presents information in an easy-to-read way, often using charts and graphs. Dashboards provide users with a quick overview of key metrics and performance indicators.
Encryption
The process of converting information or data into a code, especially to prevent unauthorized access. This is a security measure that protects your data privacy and integrity.
UI (User Interface)
The space where interactions between humans and machines occur. The goal of this interaction is to provide an effective operation and control of the machine from the human end, whilst the machine simultaneously feeds back information that aids the operators' decision-making process.
UX (User Experience)
The overall experience of a person using a product such as a website or a computer application, especially in terms of how easy or pleasing it is to use.
Compliance
The action or fact of complying with a wish or command. In a regulatory context, it involves following the laws and regulations relevant to the business operations.
