User Instructions for CSV File Import

To facilitate a seamless import of your contact data into our system, please follow the guidelines below to format your CSV file correctly. These guidelines are designed to accommodate various CSV header formats. Incorrectly formatted files may lead to unsuccessful imports or incomplete data uploads.

CSV File Format:

Your CSV file should include headers that correspond directly to the fields in our contact management system. The system is designed to recognize various header names for each field, enhancing the flexibility of the import process.

Recognized Headers:

  • First Name: 'First Name', 'firstName', 'first_name'
  • Last Name: 'Last Name', 'lastName', 'last_name'
  • Company Name: 'Company Name', 'companyName', 'Company'
  • Email: 'Email', 'email'
  • Phone Number: 'Phone Number', 'phone'
  • LinkedIn URL: 'linkedinUrl', 'LinkedIn URL'

Required and Optional Fields:

Required Fields: At a minimum, each CSV file must include:

  • First Name
  • Last Name
  • Email

Optional Fields: Additional information can be provided to enrich the contact data:

  • Company Name
  • Phone Number
  • LinkedIn URL (will be added to the social media profiles)

Nested Objects and Repeated Fields:

The utility will handle fields like phone numbers and email addresses and will default to setting types as 'Mobile' for phone numbers and 'Primary' for emails unless specified otherwise.

Boolean Fields:

For boolean fields like emailAddresses_blocked, use 'true' or 'false'.

File Encoding:

Please ensure your CSV file is encoded in UTF-8 to avoid any issues with special characters.

Template Download:

For your convenience, you can download a CSV template here that includes all the necessary headers. Simply fill in your data in the appropriate columns.

Import Process:

  1. Prepare your CSV file following the guidelines above.
  2. Navigate to the CSV Import page.
  3. Click on 'Choose File' and select your prepared CSV file.
  4. Click 'Upload' to start the import process.
  5. You will receive a notification once the import is complete or if there are any errors during the import.

Error Handling:

If errors occur, the system will provide a detailed message regarding the issue. Common errors include missing required fields, incorrect data format, or issues with file encoding.
